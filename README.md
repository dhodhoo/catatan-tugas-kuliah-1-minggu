Nama Website:

Catatan Tugas Kuliah Minggu Ini


Deskripsi Singkat:

Website ini ditujukan sebagai tempat untuk mencatat dan memantau tugas-tugas kuliah dalam periode satu minggu. Pengguna dapat melihat profil, jadwal, serta daftar tugas yang harus dikerjakan.

Navigasi Utama:

Home: Halaman utama berisi sambutan dan menu utama untuk memilih halaman lainnya.

Profile: Halaman yang menampilkan profil pengguna.

Schedule: Halaman untuk melihat jadwal kuliah minggu ini.

Tasks: Halaman untuk melihat daftar tugas kuliah minggu ini.


Fitur-Fitur:

Mode Fullscreen: Pengguna disarankan untuk mengaktifkan mode layar penuh demi pengalaman lebih baik.

Navigasi Sederhana: Menu navigasi memudahkan akses ke profil, jadwal, dan tugas.

Dark Mode Toggle: Mengganti dari mode light mode ke dark mode.

Tampilan User-Friendly: Terdapat panduan “Hello Everyone!! Selamat Datang di Website Saya” beserta pilihan menu utama dengan sekali klik.


Petunjuk Penggunaan:

Buka Website: Akses https://catatan-tugas-kuliah-1-minggu.vercel.app/.

Aktifkan Mode Fullscreen (opsional): Tekan petunjuk atau tombol fullscreen untuk pengalaman optimal.

Pilih Menu:

"My Profile" untuk melihat informasi pengguna.

"My Schedule" untuk jadwal kuliah minggu berjalan.

"My Tasks" untuk melihat daftar tugas.


Catatan:

Website bersifat sederhana dengan UI yang minimalis.

Tidak memerlukan login maupun data pribadi sensitif.

Cocok untuk mahasiswa yang ingin mencatat tugas mingguan secara praktis.


Teknologi yang Digunakan:

Frontend:

Javascript

Digunakan untuk membangun antarmuka pengguna (UI) yang interaktif dan dinamis.

CSS3

Memberikan tampilan yang responsif serta user-friendly.

HTML5

Struktur dasar halaman web.


Deployment:

Vercel

Website di-deploy secara online menggunakan Vercel.


Arsitektur:

Single Page Application (SPA)

Seluruh konten dan navigasi berjalan pada satu halaman utama tanpa reload penuh, memastikan pengalaman yang cepat dan seamless.

Tidak memerlukan login maupun data pribadi sensitif.
